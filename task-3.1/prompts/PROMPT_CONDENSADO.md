# PROMPT CONDENSADO PARA NOTEBOOKLM

Crie um podcast de 10 minutos com dois hosts discutindo este desafio técnico extraordinário.

## HISTÓRIA PARA CONTAR:

Um candidato à vaga de Analista de Monitoramento na CloudWalk (fintech unicórnio brasileira) recebeu um desafio simples: analisar CSVs de checkout e encontrar anomalias.

**A DESCOBERTA:**
Ele encontrou 3 horas consecutivas (15h-17h) com ZERO transações durante horário de pico. Z-Score de -2.8 confirmou estatisticamente. Aproximadamente 62 transações perdidas. Também identificou um spike matinal de +574% sugerindo processamento de backlog.

**O DIFERENCIAL:**
Enquanto outros entregariam um script e um gráfico, ele entregou:
- Dashboard Grafana pronto para produção
- Regras de alerta Prometheus (P1/P2/P3)
- Alertmanager para Slack/PagerDuty/Email
- Docker Compose completo
- Incident Report + Runbook + Templates Slack
- Mais de 15 arquivos profissionais

**A TRANSPARÊNCIA:**
Ele foi honesto: usou IA como parceira. MAS demonstrou que a IA é ferramenta - ele trouxe o contexto de negócio, formulou hipóteses, tomou decisões. A vaga PEDE uso de IA, e ele mostrou a parceria humano-IA perfeita.

**A MENSAGEM:**
"Bombeiros que usam código para apagar incêndios" - ele não apenas analisou dados, construiu um ECOSSISTEMA de monitoramento. Transformou um desafio simples em demonstração de criatividade, proatividade e pensamento sistêmico.

## TOM:
Dois colegas de tech genuinamente impressionados. Use os números específicos. Termine com: "Se a CloudWalk procura alguém que vê um CSV e imagina toda a infraestrutura de resposta a incidentes - encontraram."
